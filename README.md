# Satoripop Bootstrap

A responsive website for Satoripop using Bootstrap/CSS.

## Project Structure

```
├── index.html           # Main website page
├── css/                 # Custom stylesheets
│   └── main.css
├── js/                  # JavaScript files
│   └── script.js
├── bootstrap/           # Bootstrap framework (CSS & JS)
├── img/                 # Images and icons
└── fonts/               # Custom fonts
```

## Getting Started

1. Clone the repo: `git clone https://github.com/Amine-Chebil/Satoripop-Bootstrap.git`
2. Open fonts folder and install them
2. Open `index.html` in a browser
Or use Live Server in VS Code (right-click → "Open with Live Server").
